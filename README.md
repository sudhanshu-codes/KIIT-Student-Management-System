# 🎓 KIIT Student Management System

## 📌 Project Overview
The KIIT Student Management System is a web-based application developed using PHP, MySQL, HTML, and CSS. The system is designed to manage student records efficiently through an admin panel while also allowing students to view their details using their roll number.

This project was developed as an academic mini-project using XAMPP for local server and database management.

---

## 🚀 Features

### 👨‍💼 Admin Module
- Admin Login Authentication
- Add Student Information
- View Student Records
- Edit Student Details
- Delete Student Records
- Secure Admin Dashboard

### 👨‍🎓 Student Module
- Student Detail Search using Roll Number
- View Personal Academic Information

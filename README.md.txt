# Student Management System

## Project Overview
This project is a web-based Student Management System developed using HTML, CSS, PHP, MySQL, and XAMPP.

## Features
- Admin Login
- Add Student
- Edit Student
- Delete Student
- View Student
- Student Search by Roll Number

## Technologies Used
- HTML
- CSS
- PHP
- MySQL
- XAMPP

## How to Run
1. Install XAMPP
2. Start Apache and MySQL
3. Place project folder in htdocs
4. Import kiit_college.sql in phpMyAdmin
5. Open project in browser
